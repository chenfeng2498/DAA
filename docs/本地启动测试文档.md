# 大A监控本地启动测试文档

本文档记录本地系统测试的启动流程、验证方法和常见排查命令。

## 1. 环境要求

- 操作系统：Windows
- Python：3.12 或兼容版本
- 项目目录：`D:\0_CF\1-好玩的\大A监控`
- 服务端口：`8080`
- 数据库文件：`data\market.db`

确认 Python 可用：

```powershell
python --version
```

## 2. 一键启动

在项目根目录执行：

```powershell
.\start.bat
```

`start.bat` 会依次执行：

1. 初始化自选股：`python seed_watchlist.py`
2. 同步历史行情数据：`python server.py sync`
3. 启动本地服务：`python server.py`
4. 打开浏览器访问：`http://localhost:8080`

## 3. 手动启动流程

如果需要分步排查，可以按下面步骤执行。

进入后端目录：

```powershell
cd D:\0_CF\1-好玩的\大A监控\backend
```

初始化自选股：

```powershell
python seed_watchlist.py
```

同步历史数据：

```powershell
python server.py sync
```

启动服务：

```powershell
python server.py
```

启动成功后，终端会显示：

```text
大A监控服务已启动: http://localhost:8080
后台数据刷新线程已启动
```

## 4. 后台启动方式

如果希望服务在后台运行，并将日志写入文件：

```powershell
Start-Process -FilePath python `
  -ArgumentList 'server.py' `
  -WorkingDirectory 'D:\0_CF\1-好玩的\大A监控\backend' `
  -RedirectStandardOutput 'D:\0_CF\1-好玩的\大A监控\backend\server_out.log' `
  -RedirectStandardError 'D:\0_CF\1-好玩的\大A监控\backend\server_err.log' `
  -WindowStyle Hidden
```

日志文件：

- `backend\server_out.log`
- `backend\server_err.log`

## 5. 启动验证

检查 `8080` 端口是否监听：

```powershell
Get-NetTCPConnection -State Listen |
  Where-Object { $_.LocalPort -eq 8080 } |
  Select-Object LocalAddress,LocalPort,OwningProcess
```

检查首页是否可访问：

```powershell
(Invoke-WebRequest -UseBasicParsing http://localhost:8080/ -TimeoutSec 20).StatusCode
```

正常返回：

```text
200
```

检查 dashboard API：

```powershell
$r = Invoke-RestMethod -Uri http://localhost:8080/api/dashboard -TimeoutSec 40
[pscustomobject]@{
  indices = $r.indices.Count
  stocks  = $r.stocks.Count
  time    = $r.time
  summary = $r.summary
}
```

正常情况下会返回指数数量、自选股数量、更新时间和市场摘要。例如：

```text
indices stocks time     summary
------- ------ ----     -------
4       4      09:42:16 上证 3958.87，MA2500 当前估值合理；市场情绪偏强，成交量基准数据不足。
```

## 6. 停止服务

先找到监听 `8080` 的进程：

```powershell
Get-NetTCPConnection -State Listen |
  Where-Object { $_.LocalPort -eq 8080 } |
  Select-Object LocalAddress,LocalPort,OwningProcess
```

停止对应进程：

```powershell
Stop-Process -Id <OwningProcess>
```

示例：

```powershell
Stop-Process -Id 132388
```

## 7. 常见问题

### 端口被占用

如果 `8080` 已被其他进程占用，服务会启动失败。先找到占用进程：

```powershell
Get-NetTCPConnection -State Listen |
  Where-Object { $_.LocalPort -eq 8080 } |
  Select-Object LocalAddress,LocalPort,OwningProcess
```

确认可以关闭后再执行：

```powershell
Stop-Process -Id <OwningProcess>
```

### 实时行情为空

实时行情依赖东方财富接口。如果网络不可用或接口短时异常，页面仍可展示本地历史数据，但实时情绪、成交量等字段可能显示为等待数据或基准数据不足。

### 同步时间较长

`python server.py sync` 会拉取指数和自选股历史数据。首次运行或新增自选股后，耗时会更长。同步完成后会显示：

```text
同步完成
```
